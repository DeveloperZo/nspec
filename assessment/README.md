## How to use

- **Product / eng leads**: Use the assessment to prioritize roadmap and quality investments.
- **Contributors**: Use the “Prioritized recommendations” and “Pre-publish checklist” sections for concrete tasks.
- **Reviewers**: Use the pillar criteria to evaluate readiness for marketplace or internal release.

## Scope

The assessment covers:

- Reliability, security, UX, performance, maintainability, marketplace readiness, and documentation.
- No change to extension source code; this is analysis and guidance only.
